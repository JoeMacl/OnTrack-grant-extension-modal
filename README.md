# OnTrack-grant-extension-modal

This is a standalone showcase of the “Grant Extension” form I developed for the OnTrack (Doubtfire Web) admin dashboard.
It allows staff to grant assignment extensions to students using a clean Angular Material modal interface.

Features I Implemented

Angular Material modal (MatDialog)

Reactive forms with validation

Multi-student selection via <mat-select multiple>

Duration + reason fields

Frontend validation handling

Submit to backend via POST /extensions API

Error + success messages

Integration with the admin dashboard

Tech Stack

Angular

Angular Material

TypeScript

SCSS

Reactive Forms

REST API integration
